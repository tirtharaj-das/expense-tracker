# Expense Tracker

## Backend (Spring Boot)
Run:
- mvn spring-boot:run
- URL: http://localhost:8080

## Frontend (React + Vite)
Run:
- cd frontend
- npm install
- npm run dev
- URL: http://localhost:5173

## Features
- Add expense
- View all expenses
- Filter by category
- Sort by date
- Total calculation

## Tech Stack
- Java, Spring Boot, JPA
- MySQL
- React (Vite)
